# Smart Biometric Security System

A biometric access control system using fingerprint authentication with Arduino-based embedded system integration.

---

## Features

* Fingerprint Authentication
* LCD Display Interaction
* Relay-Based Door Control
* LED & Buzzer Feedback
* Fingerprint Enrollment System
* Serial Communication Monitoring

---

## Components

* Arduino UNO
* R305 Fingerprint Sensor
* Relay Module
* LCD I2C Display
* Buzzer
* LEDs
* Jumper Wires

---

## Technologies

* Arduino IDE
* Embedded C/C++
* Serial Communication
* LCD I2C Library
* Adafruit Fingerprint Library

---

## Applications

* Smart Door Lock Systems
* Attendance Systems
* Secure Access Control
* Embedded Security Systems
